# Organize your habits, make a routine and find out how efficiently you are following the routine.
 This program helps you to plan and organize your habits providing you a routine/task scheduler. Then you can track your habits everyday and statistically analyze every tasks, the steak, when you missed them and what caused you to not complete the task.

 # How to run
 First run habit_setup.py, then track your habits with habit_tracker.py. Use statistics.py and why_tasks_failed.py for analysis
